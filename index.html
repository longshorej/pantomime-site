Pantomime is a library for the Rust programming language that enables you to build concurrent and efficient programs using the Actor model.

  • Expressive
    Build asynchronous applications with ease.
  • Performant
    100M msgs/sec on a single node with linear scalability.
  • Lightweight
    Simple applications use 4MB RSS
use pantomime::prelude::*; enum Msg { Greet(String) } struct MyActor; impl Actor<Msg> for MyActor { fn receive(&mut self, msg: Msg, ctx: &mut ActorCtx<Msg>) { match msg { Greet(name) => println!("Hello {}!", name) } } }
